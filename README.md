# Logistic Regression with NumPy and Python

## Overview

This project demonstrates the implementation of Logistic Regression from scratch using NumPy and Python. The model is trained on DMV written test data to classify whether a candidate passes or fails based on two test scores.

## Objectives

* Understand the mathematics behind Logistic Regression.
* Implement the Sigmoid Function from scratch.
* Compute the Cost Function and Gradient.
* Optimize model parameters using Gradient Descent.
* Visualize model convergence and decision boundaries.
* Generate predictions and evaluate model performance.

## Technologies Used

* Python
* NumPy
* Pandas
* Matplotlib
* Seaborn

## Project Workflow

### 1. Data Loading and Exploration

* Loaded DMV written test dataset using Pandas.
* Explored the dataset structure and features.

### 2. Data Visualization

* Visualized pass and fail classes using scatter plots.
* Analyzed relationships between test scores and outcomes.

### 3. Sigmoid Function

Implemented the Logistic Sigmoid Function:

σ(z) = 1 / (1 + e^(-z))

### 4. Cost Function and Gradient

Implemented:

* Logistic Regression Cost Function
* Gradient Computation

### 5. Gradient Descent Optimization

* Updated model parameters iteratively.
* Minimized the cost function.
* Tracked convergence over iterations.

### 6. Decision Boundary

* Visualized the classification boundary learned by the model.

### 7. Predictions and Evaluation

* Generated probability predictions.
* Converted probabilities into binary classifications.
* Evaluated model accuracy.

## Results

* Successfully implemented Logistic Regression without using Scikit-Learn's LogisticRegression model.
* Optimized parameters using Gradient Descent.
* Generated predictions and visualized model performance.
